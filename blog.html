Important FAQ

What is the Document Object Model (DOM)?

DOM stands for Document Object Model.It is a programming interface through which we can create and delete elements in an HTML document and modify any type of element.And can apply different types of events on an element.Dom makes a website interactive.Dom treats the HTML document as a tree

How do you select an element from the DOM?

There are several ways to select elements from the dom in JavaScript.

1.Using getElementById()
Element can be selected by ID Name
const myElement = document.getElementById('Element Id Name')


2.Using getElementsByClassName()
Element can be selected by Class Name
const myElement = document.getElementsByClassName('Element Class Name')


3.Using getElementsByTagName()
Element can be selected by Tag Name
const myElement = document.getElementsByTagName('Element Tag Name')


4.Using querySelector()
Element can be selected by CSS Selector
const myElement = document.querySelector('CSS Selector')


5.Using querySelectorAll()
Element can be selected by CSS Selector
const myElement = document.querySelectorAll('CSS Selector')

What is event delegation in the context of the DOM, and why is it useful?

In simple terms, event delegation is the act of having a single event listener for multiple elements and making the listener work even when new elements are added.


Event delegation is an efficient technique for handling events. Event delegation An event listener is added to the parent element to handle the event and the event listener will handle the event if the event occurs on the target element.


How do you manipulate an element's attributes and styles using the DOM?

Modifying Attributes:

  • 1. Method hasAttribute()
    • Returns true if the element contains the attribute, otherwise returns false

  • 2. Method getAttribute()
    • Returns if the attribute is inside the element else returns null

  • 3. Method setAttribute()
    • It is used to set an attribute inside the element

  • 4. Method removeAttribute()
    • It is used to remove an attribute inside the element



Modifying style:

1. Style Added Using Dom
const myElement = document.getElementById('id name')
myElement.style.color = 'red'
First, we will select the element through the element selector, then we will call the style in the selector and give the name and value of the style we want.
